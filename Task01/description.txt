The first task is here to ease you in. You'll be exploring so-called 'vibe coding' — the idea is simple: build a working application using an AI-enhanced IDE and prompting techniques instead of writing code manually. You can approach it any way you like and use any prompting or agentic techniques you know. The goal is to build an application based on the requirements below.
.
Task
The task is very straightforward — you need to create a copy of our internal company leaderboard (https://ventionteamsinc.sharepoint.com/sites/edu/SitePages/CompanyLeaderBoard.aspx)
.
 Important: Do not feed any corporate data — real names, photos, or any other personal information — into AI tools. Make sure to complete the task in a way that complies with our responsible AI usage policies. 
.
Requirements
Your solution should replicate the original exactly — all UI elements, filters, sorting, and other functionality. No extra features.
Your solution should not contain any data from the original — no real names, no actual titles, no real department names. You can decide how to replace this data, but don't forget that you are building a leaderboard.
You need to deploy your solution to GitHub Pages and provide a working link to it.
Your GitHub repository should include:
source code of your application
report.md — a short write-up describing how you approached the task: what tools and techniques you used, how you handled the data replacement from the original leaderboard.
