# Approach Report

## Tools and Techniques Used
- Built using vanilla HTML, CSS, and JavaScript (no frameworks)
- AI-assisted development using GitHub Copilot in the IDE
- Visual reference from the original SharePoint leaderboard page

## Data Replacement Strategy
- All employee names are fictional (randomly generated)
- Department codes follow similar format but are entirely made up
- Activity titles follow similar naming patterns but contain no real event data
- Avatar images use placeholder services (pravatar.cc)

## Deployment
- Deployed as a static site on GitHub Pages
- No build step or server-side processing required
